## 2. Service Description

**Freedive App** is an application for breath-hold (apnea) training that provides:

- **Training timers:**
  - O2 Table (oxygen tolerance training)
  - CO2 Table (carbon dioxide tolerance training)
  - Box Breathing (4:4:4:4 pattern)
  - Rectangular Breathing (custom proportions)
  - Triangular Breathing (3-phase pattern)
  - Static Apnea Timer (STA)

- **Data management:**
  - Local storage (Hive database) - works offline
  - Cloud storage (Supabase) - syncs when signed in
  - Workout history and personal records
  - Customizable timer templates

- **Additional features:**
  - Live Activity integration (iOS 16.1+)
  - Voice announcements
  - Background audio playback
  - Google Sign-In / Apple Sign-In (optional)
  - Ad-supported (Google AdMob)

**Current version (0.0.1):** Free app with advertising support.

**Future versions:** Premium features via subscriptions are planned (will be updated in future Terms versions).
